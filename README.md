# LLM-Portfolio

A portfolio showcasing Data Science and Large Language Model (LLM) projects.

## Overview

This portfolio features a collection of machine learning and natural language processing projects, including:

- **Text Generation Model** - Transformer-based language model for creative text generation
- **Sentiment Analysis** - Deep learning model for analyzing sentiment in social media
- **Chatbot Framework** - Intelligent conversational AI with context-aware responses
- **Predictive Analytics** - Machine learning models for time series forecasting
- **Semantic Search Engine** - Vector-based search using embeddings
- **Document Summarizer** - Automatic text summarization using extractive and abstractive methods

## Technologies

- Python
- PyTorch & TensorFlow
- Hugging Face Transformers
- LangChain
- Scikit-learn & Pandas

## Getting Started

Open `index.html` in a web browser to view the portfolio, or deploy to GitHub Pages for online access.
