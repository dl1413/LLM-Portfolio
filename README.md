# LLM-Portfolio

A portfolio showcasing Data Science and Large Language Model (LLM) projects.

## Overview

This portfolio features a collection of machine learning and natural language processing projects.

## Projects

### 1. Breast Cancer Classification
**Ensemble Machine Learning for Cancer Diagnosis**

- **Notebook:** [Breast_Cancer_Classification_PUBLICATION.ipynb](./Breast_Cancer_Classification_PUBLICATION.ipynb)
- **Report:** [Breast_Cancer_Classification_Report.docx](./Breast_Cancer_Classification_Report.docx)
- **Accuracy:** 99.12% | **Precision:** 100% | **Recall:** 98.59%
- **Methods:** AdaBoost, Random Forest, XGBoost, Gradient Boosting, LightGBM
- **Techniques:** SMOTE, RFE, VIF Analysis, Cross-Validation

### 2. LLM Ensemble Textbook Bias Detection
**Multi-LLM Framework for Educational Content Analysis**

- **Notebook:** [LLM_Ensemble_Textbook_Bias_Detection.ipynb](./LLM_Ensemble_Textbook_Bias_Detection.ipynb)
- **Report:** [LLM_Textbook_Bias_Detection_Report.docx](./LLM_Textbook_Bias_Detection_Report.docx)
- **LLM Ensemble:** GPT-4, Claude-3-Opus, Llama-3-70B
- **Inter-Rater Reliability:** Krippendorff's Î± = 0.84
- **Methods:** Bayesian Hierarchical Modeling, Friedman Test, Wilcoxon Post-hoc

## Technologies

- **Machine Learning:** scikit-learn, XGBoost, LightGBM
- **Bayesian Analysis:** PyMC, ArviZ
- **Data Science:** NumPy, Pandas, Matplotlib, Seaborn
- **LLM APIs:** OpenAI, Anthropic, Together AI
- **Report Generation:** python-docx
